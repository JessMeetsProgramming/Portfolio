Project One

Waipukurau Pizzeria

The Waipukurau website was a project for my studies, it was primarily focused on the back-end database design. I started with designing the front-end HTML pages with the forms that would provide the basis for the CRUD system - which included booking and ordering pages. These pages were integrated with an existing database using PHP and MySQL. A simple user access system was added using PHP sessions to control admin vs customer access to pages and to allow users to create accounts and manage their orders and bookings.

Form validation was written in PHP before making any queries to the database and prepared statements were used to prevent any potential SQL injection attacks. Extensive testing was performed on all the database functions, I used Selenium IDE to create test suites to automate the process for maintenance purposes. We were provided with all the assets and a template for the design of this website, so it was just a matter of integrating the database pages and making sure the styling matched up.

The Haukai Restaurant

Haukai was also an assignment, continuing in the restaurant theme - we were given a brief and were tasked with creating a website using HTML and CSS, I also found use for some Javascript. There was plenty of opportunity to be creative with this project as I was responsible for the design from conception through to completion, I played around with fonts, images and information architecture to display the relevant content.

Responsive design and performance on slower networks were accounted for when considering the user experience, as many customers would likely access the website from their mobile devices. Google APIs were implemented for dynamic content such as maps and forms. After completing the website we were asked to redo the website styling using a web framework to simplify code maintenance, the website is currently styled using W3.CSS.
